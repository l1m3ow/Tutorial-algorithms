## Что такое O-большое (которое будет встречаться в этом пособии)?

O-большое — это математическая нотация, которая описывает скорость роста времени выполнения алгоритма в зависимости от объема входных данных. Она позволяет сравнить эффективность алгоритмов. Важно понимать, что O-большое рассматривает худший сценарий (верхнюю границу) количества операций.

Типичные примеры сложности:

* O($log_2n$) — логарифмическая сложность (очень быстрый алгоритм).

* O($n$) — линейная сложность (быстрый алгоритм).

* O($n * log_2n$) — линейно-логарифмическая сложность (средняя эффективность).

* O($n^2$) — квадратичная сложность (медленный алгоритм).

* O($n!$) — факториальная сложность (очень медленный алгоритм).
